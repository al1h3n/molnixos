### Changing variables for your needs:
Open variables.nix and change everything you need.
For instance, location of hyprland configuration.

### Important things:
1. Add flake.nix and flake.lock (as well as any flakes file) to git:
`git init`
`git add .`
`git commit -m "flakes"`


### Where files will be saved?
Configurations: `/etc/nixos"` | shared dotfiles: `/etc/nixos/molniux"`

### When to install?
Install MolnixOS after you had installed nixOS.
